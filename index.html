IMDbPY is a Python package for retrieving and managing the data of the IMDb movie database about movies, people, and companies.

Downloads Read the Docs

  • Written in pure Python 3.
  • Platform-independent.
  • Can retrieve data from both the IMDb's web server and a local copy of the whole database.
  • Released under the terms of the GPL 2 license.
  • A simple API.
  • IMDbPY is used by many other projects. Curious about that?

Code example


from imdb import IMDb

# create an instance of the IMDb class
ia = IMDb()

# get a movie and print its director(s)
the_matrix = ia.get_movie('0133093')
print(the_matrix['director'])

# show all information sets avaiable for movies
print(ia.get_movie_infoset())
